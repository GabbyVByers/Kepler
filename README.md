# Kepler
Kepler is an interactive 3D simulation of the major planets and moons of the solar system. Kepler is intended to serve as a prototype for a KSP-inspired game that enables players to explore space using realistic rockets and orbital maneuvers. Kepler is built on top of a custom library for 3D application development called Violet.  Violet supports double-precision floating-point arithmetic throughout its entire API, enabling the kind of large-scale world building required by Kepler. Kepler does not yet have an official release date.
